Introduction

The IP Address to Location API is organized using REST architectural patterns. You can estimate the geolocation, country, and city from a given IP address (IPv4 only). The API is free to use. It requires no authentication or registration.

End Point Details

Description

Returns a location estimate (country, state, city, post code, latitude and longitude coordinates) when given a valid IP address.

API Method Name

srvs_getiploc

Base URL

https://matkarma.in/logv/srvs_getiploc.php?ip=

Example API Call

https://matkarma.in/logv/srvs_getiploc.php?ip=185.177.57.20

Notes
  1. This service is in beta. It's hosted on a server with 2 GB of RAM, so expect limited load tolerance. Send us an email and if you plan to use it beyond hobbyist projects (5000+ calls per hour).
  2. No authentication required.
  3. IP address (Only IPv4 supported currently) is a required parameter.
